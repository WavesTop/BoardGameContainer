# 德州扑克基础好友局 v1

## 适用范围

- 2～8 人好友房，仅使用不可兑换的虚拟筹码。
- 每位玩家初始 1000 筹码，小盲 10、大盲 20，庄位每手顺时针移动。
- 使用 52 张标准扑克牌，无大小王。

## 单手流程

1. 下小盲和大盲，每位有效玩家发两张底牌。
2. 依次进行翻牌前、翻牌、转牌和河牌下注圈；翻牌发三张公共牌，之后各发一张。
3. 玩家可按局面弃牌、过牌、跟注、加注或全下。加注至少达到上一次完整加注的增量；不足最小加注的全下不会降低最小加注额。
4. 只剩一位未弃牌玩家时立即获得底池；否则河牌下注结束后摊牌。
5. 从七张可用牌中选择最强五张，按高牌、一对、两对、三条、顺子、同花、葫芦、四条、同花顺比较，同牌型逐级比较踢脚。
6. 全下金额不同时按实际投入建立主池和边池；每个池只在有资格的玩家中比较，平分后的零头按座位顺序发放。

## 人机策略

人机使用底牌质量、当前成牌、底池赔率、跟注占剩余筹码比例和少量可控诈唬共同决策。强牌会价值加注，边缘牌会按赔率跟注，弱牌面对压力会弃牌；无下注时可以过牌，也可能用合适牌力主动下注，不固定执行“只跟注”或“只加注”。
