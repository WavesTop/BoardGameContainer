# 贵州麻将基础好友局 v1

## 规则边界

贵州各地细则差异较大。本版本冻结为可完整游玩的基础好友局，不把贵阳捉鸡、冲锋鸡、责任鸡等计分扩展混入基础状态机；后续可在同一摸打与响应窗口上增加独立计分预设。

- 固定 4 人，使用万、筒、条各 1～9 共 108 张牌，不使用字牌和花牌。
- 庄家 14 张起手，其余玩家 13 张；庄家先打，之后顺时针摸一打一。
- 不允许吃，可碰、明杠、暗杠和补杠；杠后从牌墙另一端补牌。
- 支持点炮、自摸、一炮多响、普通四组面子加一对将和七对。
- 响应优先级为胡高于杠、杠高于碰；同优先级按出牌者之后的座次处理。多人同时选择胡时全部成立。
- 无人响应后下家摸牌；牌墙耗尽且无人胡牌则流局。

## 人机策略

人机会优先胡牌，在牌墙充足时合理杠牌；碰牌前比较副露带来的结构收益，收益不足会选择过。弃牌通过刻子、对子、相邻搭子、嵌张搭子和孤张结构评分选择，优先保留成组机会，不会机械打出刚摸到的牌。
