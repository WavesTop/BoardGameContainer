# 斗地主与麻将体验对齐清单

本清单用于对齐成熟产品的流程密度、信息层级、操作反馈和牌桌可读性，不复制第三方商标、角色、插画、牌背纹样、活动包装或其他独特美术资产。

## 斗地主

公开流程基线：模式选择 → 场次/好友房 → 发 17 张手牌并扣 3 张底牌 → 叫/抢地主 → 地主获得底牌 → 不加倍/加倍/超级加倍 → 地主先出 → 提示/不出/出牌 → 炸弹与火箭翻倍 → 阵营结算。

牌桌必须同时呈现：

- 三名玩家的人物头像、昵称、地主/农民身份和剩余牌数。
- 三张底牌；叫地主前隐藏，确定地主后亮出。
- 当前阶段、当前行动者、底分和实时倍数。
- 手牌选中抬升、牌型反馈、提示、不出、出牌和托管。
- 最后一手牌、出牌者和牌型；一轮结束后恢复自由领出。
- 结算中的获胜阵营、倍率与得分构成。

## 麻将

公开流程基线：玩法选择 → 场次/好友房 → 洗牌/发牌 → 庄家起手 → 摸牌与弃牌 → 响应窗口 → 碰/杠/胡/过 → 补张 → 自摸/点炮/流局 → 结算。

牌桌必须同时呈现：

- 四向人物座位、昵称、庄家标记、手牌数、副露数与行动状态。
- 四边牌墙、四家河牌、碰杠副露和中央方位/巡数/余牌。
- 本地手牌、选中抬升、智能理牌、杠、胡和打出。
- 响应窗口仅显示当前合法操作，并以“胡”作为最高视觉优先级。
- 结算中的胡牌者、自摸/点炮/流局和再次开局入口。

## 原创视觉约束

- 人物使用项目自有原创角色，不沿用第三方角色脸型、服装、姿势或命名。
- 牌桌采用深蓝夜景与青绿桌布两套主题，但不复刻第三方独特场景、纹样或品牌配色组合。
- 纸牌与麻将牌面优先保证小屏识别和操作命中，不复制第三方专有牌面资产。
- 所有关键操作保持至少 48 × 48 逻辑像素的触控区域，16:9 横屏首版按 1600 × 900 设计。
